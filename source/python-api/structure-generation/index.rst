Structure Generation
====================

Our platform provides access to the RFdiffusion model (`Watson et al., 2023 <https://www.nature.com/articles/s41586-023-06415-8>`_). This model provides structure generation capabilities using diffusion, which can be used for workflows like de novo binder design.
