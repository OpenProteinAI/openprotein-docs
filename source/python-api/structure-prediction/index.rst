Structure Prediction
====================

Our Python API provides access to open source models for structural prediction - including ESMFold, (`Lin et al., 2023 <https://www.science.org/doi/10.1126/science.ade2574>`_) and AlphaFold2 (`Jumper et al., 2021 <https://www.nature.com/articles/s41586-021-03819-2>`_). Use these structure prediction models to derive meaningful representations of your protein sequence.
